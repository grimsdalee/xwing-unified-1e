# Rules Conversion Strategy

## Reuse

-   Object spawning
-   Base/peg attachment
-   Model attachment
-   Object lifecycle
-   Multiplayer-safe scripting
-   Save/load behaviour

## Convert

-   Pilot abilities
-   Upgrade behaviour
-   Damage deck
-   Tokens
-   Squad building
-   Initiative / Pilot Skill
-   Factions
-   Epic rules

## Remove

-   Medium bases
-   Scenario mechanics
-   Objectives
-   ROAD (Random Order After Dials)
-   Other 2.5-only gameplay systems

# Asset Pipeline

Goals:

- Avoid dead Steam CDN URLs.
- Prefer stable GitHub-hosted assets.
- Track every mesh, texture, collider, and card source.
- Validate URLs before release.

Example asset manifest:

```json
{
  "id": "t65-xwing-red",
  "type": "ship-model",
  "mesh": "",
  "diffuse": "",
  "collider": "",
  "source": "",
  "notes": ""
}
```

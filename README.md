# X-Wing Miniatures Unified – First Edition

A community restoration/rebuild of **X-Wing Miniatures 1st Edition** for Tabletop Simulator, using the maintained Unified 2.0/2.5 architecture as the technical foundation while preserving 1E gameplay, cards, tokens, dials, damage decks, and factions.

## Project goals

- Preserve the full X-Wing Miniatures 1st Edition experience.
- Use stable, maintainable assets rather than dead Steam CDN links.
- Keep the project data-driven where possible.
- Build reproducible releases for Tabletop Simulator.
- Document every asset, script, and migration decision.
